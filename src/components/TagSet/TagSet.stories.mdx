## Explore Component Props

An "explore" `TagSet` variant is used to display content that can help a user
start a search and promote exploration. These tags are not dismissible and should
be used as links to other pages with the content highlighted on the new page.

Notes:

- The `isDismissible` and `onClick` props are not used in this variant.

### Explore `tagSetData` Labels

The `tagSetData` prop is an array of objects that contain `iconName` and `label`
properties. The `iconName` is the name of the icon to display through the
Reservoir `Icon` component and are optional. The `label` _MUST_ be a React
element; typically, links are expected to be used for the "explore" variant.

```tsx
const tagSetData = [
  { iconName: "arrow", label: <a href="#">Abbey Road</a> },
  { iconName: "actionLaunch", label: <a href="#">The Beatles</a> },
  { iconName: "alertWarningOutline", label: <a href="#">Revolver</a> },
  {
    iconName: "check",
    label: <a href="#">The Beatles Sgt. Pepper's Lonely Hearts Club Band</a>,
  },
];
```

## Filter Component Props

A "filter" `TagSet` variant is used to display keywords that a user selected to
filter down the scope of the content on a page. For example, in a search results
page, the filters a user selects would be displayed back to the user through
the "filter" `TagSet` variant. These tags can be dismissed by the user and the
scope of the content should be updated accordingly -- but the update to display
the proper filters and updated results are the responsibility of the
consuming application.

Notes:

- The `isDismissible` prop is optional and the `onClick` prop is
  required for this variant.
- The `onClick` prop will only fire when `isDismissible` is true and the user
  clicks a tag.
- Icons will not be rendered when `isDismissible` is true. In this case, the "X"
  close icon will be rendered instead.

### Filter `tagSetData` Labels

The `tagSetData` prop is an array of objects that contain `iconName` and `label`
properties. The `iconName` is the name of the icon to display through the
Reservoir `Icon` component and are optional. The `label` _MUST_ be a string for
the filter type.

```tsx
const tagSetData = [
  { iconName: "arrow", label: "Abbey Road" },
  { iconName: "actionLaunch", label: "The Beatles" },
  { iconName: "alertWarningOutline", label: "Revolver" },
  {
    iconName: "check",
    label: "The Beatles Sgt. Pepper's Lonely Hearts Club Band",
  },
];
```

## Accessibility

All proper tags in the "explore" and "filter" `TagSet` variants are accessible.
For the "explore" variant, tags are expected to be set as links and are therefore
focusable. For the "filter" variant, tags are rendered as buttons to perform
in-page clicking behavior.

Resources:

- [W3C WAI-ARIA Button](https://www.w3.org/WAI/ARIA/apg/patterns/button/)
