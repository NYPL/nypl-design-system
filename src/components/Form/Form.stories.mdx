# Form

<Description of={Form} />

The `Form` component renders a standard `<form>` element and should be used to handle spacing for child input fields. Use the `FormRow` and `FormField` child-components to arrange your input fields accordingly.

Multiple `FormField` components can be placed in a parent `FormRow` component. However, a `FormField` component does not have to placed within a `FormRow` component. `FormField` components can stand alone within `Form`.
