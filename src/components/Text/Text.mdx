## Overview

<Description of={TextStories} />

The `Text` component renders a basic `<p>` element and can be used to render
text for differing content types, including multiple subtitle variants.

## Accessibility

The `Text` component renders paragraph `p` HTML elements which are inherently
accessible and will be read by screen readers.

Resources:

- [Chakra UI Text](https://chakra-ui.com/docs/components/typography/text)

## Updated Typographic Styles

New typographic styles were introduced in June 2023, adding `subtitle1`,
`subtitle2`, `overline1` and `overline2`. As part of that update, the `tag` and
`mini` styles have been deprecated.

For usage recommendations of each of the `size` options, please refer to the
[Typography Style
Guide](../?path=/docs/style-guide-typography--docs#text-display-sizes).

### Recommended Options

`body1`, `body2`, `caption`, `subtitle1`, `subtitle2`, `overline1`, `overline2`
