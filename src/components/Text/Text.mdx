## Overview

<Description of={TextStories} />

The `Text` component renders a basic `<p>` element.

As its name indicates, the `size` prop controls the size of the text rendered in
the component. Using the `size` prop, the `Text` component can be used to render
text for differing content types, including various forms of subtitles.

## Accessibility

The `Text` component renders paragraph `p` HTML elements which are inherently
accessible and will be read by screen readers.
