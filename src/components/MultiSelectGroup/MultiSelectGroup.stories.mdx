## Accessibility

The `MultiSelectGroup` renders a group of `MultiSelect` components that are wrapped in
a `<fieldset>` element. The `<fieldset>` element renders a `<legend>` element that
can be visually hidden through the `showLabel` prop. If the `<legend>` is hidden the `aria-label`
attribute will be added to the `MultiSelectGroup` component.

## Layout Patterns

The width of the MultiSelect components can be passed through the `multiSelectWidth` prop.
On screen width under 600px width the `layout` is automatically set to "column" and the `multiSelectWidth` to "full".
