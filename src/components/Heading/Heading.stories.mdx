# Heading

<Description of={Heading} />

A `Heading` component will render a standard HTML `<h>` tag (1-6). The heading's text can be passed in through a `text` prop or as a child.

<Preview withToolbar>
  <Story
    name="Basic"
    args={{
      id: "heading1",
      level: 1,
      text: "Heading  1",
    }}
  >
    {args => <Heading {...args} />}
  </Story>
</Preview>
