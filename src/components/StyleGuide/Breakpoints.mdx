## General Information

The Reservoir Design System (DS) adheres to four global breakpoints for
responsive layouts and typography.

- Small Mobile: less than 480px
- Large Mobile: less than 768px
- Small Tablet: less than 1024px
- Large Tablet: less than 1280px
- Desktop: greater than or equal to 1280px

## CSS Variables

The following CSS variables are available through the DS.

<Source
  code={`
// Chakra Syntax
--nypl-breakpoint-small: 480px;
--nypl-breakpoint-medium: 768px;
--nypl-breakpoint-large: 1024px;
--nypl-breakpoint-xl: 1280px;

// NYPL Semantic Syntax
--nypl-breakpoint-smallMobile: 480px;
--nypl-breakpoint-largeMobile: 768px;
--nypl-breakpoint-smallTablet: 1024px;
--nypl-breakpoint-largeTablet: 1280px;
`}
language="css"
/>

## SCSS Variables

If you are using SCSS and imported the DS `/dist/resources.scss` file, then you
can use the following SCSS variables in your media queries or SCSS `breakpoint`
function (if used).

<Source
  code={`
$nypl-breakpoint-small: 480px;
$nypl-breakpoint-medium: 768px;
$nypl-breakpoint-large: 1024px;
$nypl-breakpoint-xl: 1280px;
$nypl-max-width: $nypl-breakpoint-xl;
`}
  language="scss"
/>
