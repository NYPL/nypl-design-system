<Description of={Card} />

Cards are contained units of information that can stand independently from other elements for context. While a Card is a self-contained unit, Cards are often grouped together and displayed in some type of grid layout.

## Custom Image Component

Instead of passing a path for the `src` prop, you can pass a custom image component in using the `component` prop.

<Preview withToolbar>
  <Story name="Custom Image Component">
    {(args) => (
      <Card>
        <CardImage
          alt="Alt text"
          imageAspectRatio={CardImageRatios.SixteenByNine}
          component={
            <Image src="https://placeimg.com/400/400/animals" alt="Alt text" />
          }
        />
        <CardHeading level={2} id="heading1">
          Card Component with Custom Image Component
        </CardHeading>
        <CardHeading level={4} id="heading2">
          Secondary Heading
        </CardHeading>
        <CardContent>
          Nulla vitae elit libero, a pharetra augue. Lorem ipsum dolor sit amet,
          consectetur adipiscing elit. Aenean lacinia bibendum nulla sed
          consectetur. Vestibulum id ligula porta felis euismod semper. Cras
          justo odio, dapibus ac facilisis in, egestas eget quam. Sed posuere
          consectetur est at lobortis.
        </CardContent>
        <CardActions>
          <Link type={LinkTypes.Button} href="blah">
            Reserve
          </Link>
          <Link href="#url" type={LinkTypes.Forwards}>
            View Book Details
          </Link>
        </CardActions>
      </Card>
    )}
  </Story>
</Preview>
