# Hero

<Description of={Hero} />

The `Hero` component is used to display a full width banner at the top of a page.
The `Hero` will contain a required `h1` page title and may also include optional
descriptive text and images.

## Color Variations for Secondary Hero

The background color for the title bar in `HeroTypes.Secondary` is determined
by the placement of the `Hero` component within a comsuming app. The styling for
`Hero` is constructued to use classes of the `Hero` component's parent elements
as a determinant for the title bar background color. Below are the four color
variations available based on the consuming React app.

```jsx
<div className="nypl--books-and-more">
  <Hero {...props} />
</div>
```
